Totp-cli

TOTP CLI tool written in Go.

Download this project as a .zip file Download this project as a tar.gz file

This is a simple TOTP (Time-based One-time Password) CLI tool. You can manage your accounts with namespaces and protect your data with a PIN (aka password).
